SubSpec is an open-source LLM inference framework tailored for tree-based speculative decoding (SD) research.

- Supports multiple offloading pipelines and target/draft model quantization strategies.
- Includes an optimized tree-based SD pipeline with full `torch.compile` support for maximal speedup.
